WEB-SAFE FONTS

These are fonts that work on most computers. If a web-safe font is either your specified font or one of your fallbacks -- fonts the computer displays if the first font specified is not available on the computer -- your webpages will display correctly on the majority of computers.

Examples of fonts that exist on most computer systems:

  • Arial
  • Helvetica
  • Times
  • Times New Roman
  • Courier New
  • Courier

With mobile devices, however, the list of web-safe fonts is far smaller.

Font Size

Font-size is the size of font. Font sizes can be measured in pixels, points, em, or percent. The font-size selector will change the size of the font of an element. Changing the font size may change the size of other items since it is used to compute the value of em and ex length units.

Examples of using font size

  • h1 {font-size:50px;}
  • h1 {font-size:2.5em;}
