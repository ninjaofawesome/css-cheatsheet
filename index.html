Font Style

The font-style CSS property allows you to select italic or oblique font-faces within your chosen font family.

Italic forms are generally cursive, while oblique forms are generally sloped.

Values:

  • normal: selects a font that is classified as normal
  • italic: selects an italic form; if that is not available, selects an oblique form
  • oblique: selects an oblique form

Example syntax:

p { font-style: italic }

Web-Safe Fonts

These are fonts that work on most computers. If a web-safe font is either your specified font or one of your fallbacks -- fonts the computer displays if the first font specified is not available on the computer -- your webpages will display correctly on the majority of computers.

The complete list of web-safe fonts can be found here: CSS Font Stack.

With mobile devices, however, the list of web-safe fonts is far smaller, at least according to this website:

  • Arial/Helvetica
  • Courier/Courier New
  • Georgia
  • Times/Times New Roman
  • Trebuchet MS
  • Verdana

If you want to use a font that isn't on the list of web-safe fonts, you can use one of the tools we discuss below.

Font Support

When declaring a font family to use on a site, the family of fonts is chosen in a heiarchy. The first choice font will be listed first (it could be an included font), followed by the next choice if that font does not want to render, followed by more choices, then followed by a final choice available to most computers everywhere.

The two most general families of fonts to choose from are serif fonts or sans-serif fonts. Beyond that there are a small variety of web safe fonts to choose from. (Please refer to this page for a full listing.) The designer can then choose their first choice of fonts by using the @font-face rule in most browsers today.
